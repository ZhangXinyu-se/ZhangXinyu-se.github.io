💻 New Project！✌️Progressing....
  • Pest Detection System based on YOLOv8 (2025)
    Situation: Developed an intelligent pest detection system for smart agriculture to address low accuracy and inefficient annotation in field scenarios.
    Task: Led the pest detection module, responsible for dataset construction and YOLOv8 model optimization.
    Action: Applied attention mechanism, local feature enhancement, adaptive feature fusion, illumination-invariant feature extraction, model structure refinement, and training strategy improvement; built a self-developed annotation tool.
    Result: Improved model mAP from 10.83% to 67.13% (6.2× enhancement); boosted annotation efficiency by over 3×; won National Third Prize in China Collegiate Computing Contest.
  • Multi-threaded Web Data Collection & Mining System (2024)
    Situation: Traditional single-threaded crawlers are limited to simple tasks, lacking concurrency support, task scheduling, and system scalability.
    Task: Designed and built a fully functional, multi-task web data collection, mining, and analysis system to support complex data acquisition scenarios.
    Action: Developed a Python-based multi-threaded concurrent crawler with independent task scheduling, data parsing, cleaning, storage, and visualization modules.
    Result: Supported stable parallel data collection and comprehensive data processing; successfully crawled and processed 10,000+ valid data entries; delivered an extensible and practical data mining system.
    Demo: https://github.com/ZhangXinyu-se/multithread-data-collection-mining-system
